Tutorials
=========

The *clkhash* library can be used via the Python API or the command line tool `clkutil`.
The tuturials `tutorial_api.ipynb` and `tutorial_cli.ipynb` show example linkage workflows for
both cases.

With linkage schema version 3.0 *clkhash* introduced different comparison techniques for feature values.
They are described in the tutorial `tutorial_comparisons.ipynb`.

running the tutorials
^^^^^^^^^^^^^^^^^^^^^
You can download the tutorials from `github <https://github.com/data61/clkhash/tree/master/docs>`_.
The dependencies are listed in `doc-requirements-other.txt` and `doc-requirements-anonlink.txt`.
